# PDFEncry

這個 PDFEncryed專案可將 PDF 依傳入的檔案進行加密後產出加密的 PDF 檔案,
使用 iTextSharp 5.0.5 版本 (http://itextpdf.com/).
請參考 http://itextpdf.com/terms-of-use/ 的授權資訊.

PDFEncry is licensed as [AGPL] software.


# 安裝方式
執行 C:\Windows\Microsoft.NET\Framework64\v4.0.30319\InstallUtil.exe EncoderPDFWindowsService.exe , 
並啟動服務 EUPdfEncryService
